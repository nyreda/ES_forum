# Earshit

## 介绍
一个用Django实现的论坛。

这个论坛的原名是 Earshit ，作者在2026-01-07编辑的时候觉得这个名字不太雅观，于是改成了 ES Forum。

#### 为什么最初叫 Earshit ？
这个项目的灵感来自于wezy和另一位初中同学给班级做的《二班史册》网站，因此这个项目最初也叫二班史册，简称二史。二史谐音耳屎，于是叫Earshit。

#### 论坛为什么最终没有上线？
因为我懒得管理。

## 功能
用户系统
- 注册登录：支持邮箱验证码注册、图形验证码登录、以及基于 Session 的登录状态保持。

- 账户安全：提供基于邮箱验证的密码找回功能。

- 个人中心：支持查看及修改个人资料，提供账号“注销”功能。

论坛核心功能

- 内容发布：支持用户发布帖子及对帖子进行评论。

- 浏览与搜索：支持按分类（公告、推荐、最新、热门）查看，以及基于标题和内容的全文搜索。

- 个人主页：展示用户发布的帖子，支持博主管理自己的内容。

- 内容保护：未通过审核的帖子仅作者和管理员可见。

管理员

- 用户管理：管理员可对用户信息进行增删改查。

- 内容审核：完善的审核流，支持对帖子和评论进行“通过”或“拒绝”操作。

- 系统日志：维护版本更新日志，记录系统动态。

## 运行环境

- Django3.2.9
